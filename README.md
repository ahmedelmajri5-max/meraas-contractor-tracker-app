# منظومة تتبع المقاولين - شركة ميراس (MERAAS Contractor Tracker)

منظومة ويب متكاملة مبنية بلغة بايثون (Python) وواجهة أمامية تفاعلية حديثة (Vanilla JS & CSS) لتتبع أوامر العمل والمستحقات والمدفوعات للمتعهدين والمقاولين بالربط المباشر مع نظام Odoo 16 عبر XML-RPC API.

## 🚀 كيفية التشغيل محلياً (Quick Start)

1. تأكد من تثبيت لغة بايثون 3 على جهازك.
2. قم بتشغيل الخادم الخلفي:
   ```bash
   python server.py
   ```
3. افتح المتصفح وتوجه إلى الرابط التالي:
   **[http://127.0.0.1:8000](http://127.0.0.1:8000)**

---

## 🔒 حماية البيانات الحساسة (Security Alert)

> [!WARNING]
> يحتوي هذا المجلد على ملف `.gitignore` لمنع رفع ملفات البيانات الحساسة (`database.json` و `contractors_accounts.csv`) التي تحتوي على كلمات مرور وحسابات المقاولين.
> يرجى عدم إزالة هذه الملفات من الـ `.gitignore` قبل الرفع إلى GitHub.
> كما يوصى بإخفاء مفتاح Odoo API Key في ملف `.env` بدلاً من كتابته بشكل مباشر في كود `server.py` عند الإنتاج.

---

## 📂 هيكل المجلدات (Project Structure)

- `server.py`: خادم الويب والمتحكم في الاتصال والربط مع Odoo.
- `generate_and_map_contractors.py`: سكربت مساعد لربط المقاولين تلقائياً وإنشاء حسابات الدخول لهم.
- `public/`: مجلد ملفات الواجهة الأمامية (HTML, CSS, JS، وشعار الشركة).
- `.gitignore`: ملف استثناء الملفات الحساسة والمؤقتة من الرفع للجيت هاب.
